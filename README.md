# local-directory
Local Stores & Services Directory
================================

Tech:
- React + Vite
- Cloudflare Pages
- Cloudflare R2

Features:
- India-only hierarchy
- Mandal-wise JSON loading
- Paid promoted stores
- One template per store/service type
- Images from R2

Deploy (Cloudflare Pages):
Build command: npm run build
Output directory: dist

Bind R2 bucket as:
R2_BUCKET
